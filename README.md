# Hydraulic Fracturing Parameter Optimization for Shale Reservoirs

## Overview
This project presents a simplified hydraulic fracturing design and optimization tool based on petroleum engineering principles.  
The model evaluates the impact of pump rate and fluid properties on fracture geometry and treatment pressure using PKN-based correlations.

## Objectives
- Estimate net fracturing pressure
- Analyze fracture width and length
- Perform sensitivity analysis on pump rate
- Visualize treatment response

## Inputs
- Injection rate (bpm)
- Fluid viscosity (cp)
- Young’s modulus (GPa)
- Fracture height (m)

## Methodology
- Simplified PKN fracture model
- Analytical correlations for pressure and geometry
- Python-based numerical analysis

## Results
- Pressure vs Pump Rate
- Fracture Width & Length vs Rate

## Tools & Skills
- Python (NumPy, Matplotlib)
- Petroleum Engineering Fundamentals
- Hydraulic Fracturing Design
- Data Visualization

## Disclaimer
This model is for educational and screening-level analysis only.
